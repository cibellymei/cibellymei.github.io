COMO ADICIONAR IMAGENS AO MEU CÓDIGO HTML5?

♥ IMAGENS INTERNAS ♥

♥ 1º Passo: Crie uma pasta ♥

☀ Para você adicionar uma imagem ao seu código, é bem simples, crie uma pasta imagem ou com o nome de sua preferência dentro da pasta onde seu código está sendo criado.


♥ 2º Passo: Abrindo a tag ♥

☀ Abra a tag img e dentro do src, pressione ctrl + espaço, escolha a pasta que você criou e selecione a imagem que você deseja incluir em seu código, o mesmo serve para subpastas e lembre-se de incluir uma breve descrição dentro do alt.

☀ Parabéns, você conseguiu adicionar imagens hospedadas em servidor próprio ao seu código HTML5, para incluir imagens externas, siga o próximo passo a passo.

♥ IMAGENS EXTERNAS ♥

♥ 1º Passo: Escolha uma imagem livre de direitos autorais ♥

☀ É importante que a imagem não tenha direitos autorais ou seja de autoria própria para evitar problemas com a justiça, copie o link dessa imagem e adicione seguindo o passo a seguir.

♥ 2º Passo: Utilize a tag img src novamente ♥

☀ Dessa vez, você colará o link da imagem dentro do src, não esqueça de colocar a breve descrição da imagem no alt.

♥ IMAGENS DINÂMICAS ♥

☀ Experimente abrir esse site em outros dispositivos ou apenas aumente e diminua o tamanho do seu navegador.

☀ Para fazer isso, basta utilizar a tag picture, abrir a tag source e específicar o tamanho desejado, preencha o srcset com o caminho das imagens desejadas, especifique o tipo de item que você irá adicionar com o type.
